# Test récapitulatif du cours
Ce test récapitulatif du cours contient 17 questions et vaut pour 30 % de la note finale. Nous vous recommandons d’y répondre après avoir visionné et consulté l’ensemble du matériel de ce module et avoir fait les tests 1 à 4.

Pour chaque question, vous avez droit à deux essais (sauf dans le cas d'une question avec deux choix de réponses). Après avoir sélectionné votre réponse, cliquez sur le bouton Soumettre pour voir si votre réponse est correcte (ou sur Sauvegarder si vous voulez y revenir plus tard). Si votre réponse est correcte, un crochet vert apparaît et vous obtenez les points associés. Si une croix rouge apparaît, cela signifie que la réponse est incorrecte (ou partiellement incorrecte pour les questions à choix multiples), vous pouvez alors essayer une autre réponse. Dans ce cas, cliquez sur Soumettre pour envoyer votre réponse finale.

Vous pouvez voir votre score final au test dans l’onglet Progression.

Veuillez à respecter le code d'honneur EdX au sujet des tests.
